# E-dit-Desktop

**E-dit** — это профессиональный двухпанельный инструмент для форматирования, очистки и редактирования текстов. Приложение построено на современном веб-стеке и оптимизировано для работы с большими объемами текстовых данных без задержек ввода.

---

## 🛠 Технологический стек

*   **Фреймворк:** React 18 (TypeScript)
*   **Сборщик:** Vite 5
*   **Стилизация:** TailwindCSS & PostCSS (аккуратный темный интерфейс с эффектом стеклянного размытия и микроанимациями)
*   **База данных:** Dexie.js (обертка над IndexedDB для локального сохранения заметок, пресетов и сессий)
*   **Иконки:** Lucide React
*   **Десктопная интеграция:** Tauri 1.5 (для нативной работы с файловой системой)

---

## 🚀 Основные возможности

1.  **Двухпанельный режим (Split View):** Возможность независимого редактирования двух текстов на одном экране с быстрым переключением фокуса и горячими клавишами.
2.  **Базовое форматирование:** Инструменты для удаления лишних пробелов, неразрывных пробелов (NBSP), пустых строк, форматирования регистров и знаков препинания.
3.  **Специфическая обработка текстов Suno:** Очистка, отбивка, форматирование тегов песен/структуры, разделение метаданных от текста.
4.  **Цепочки команд (Пресеты):** Создание и сохранение собственных цепочек форматирования, выполняющих несколько операций за один клик.
5.  **Анализатор символов:** Панель с подсчётом уникальных спецсимволов по частоте встречаемости с возможностью их быстрого удаления в один клик.
6.  **Менеджер заметок:** Автономная боковая панель с возможностью создания, обновления и быстрого переключения заметок, синхронизированная с локальной БД.

---

## ⚡ Оптимизации производительности

Приложение спроектировано для максимальной отзывчивости интерфейса:
*   **Дебаунс статистики:** Вычисление объёмных метрик текста (кол-во слов, строк, абзацев) выполняется с задержкой в 300 мс, чтобы не перегружать поток выполнения на каждый клик клавиатуры. Общий счётчик символов при этом обновляется мгновенно.
*   **Дебаунс анализатора символов:** Подсчёт частоты символов и парсинг текста обновляется через 300 мс после окончания ввода, предотвращая лаги при редактировании больших текстов (100k+ символов).
*   **Дебаунс автосохранения:** Запись состояния редактора в IndexedDB происходит через 1 секунду после прекращения набора текста.
*   **Оптимизация бандла:** React и Dexie вынесены в отдельные JS-чанки (`manualChunks`) для эффективного кэширования браузером и ускорения первой отрисовки (FCP).

## 📦 Быстрый старт

Убедитесь, что у вас установлена Node.js.

### Установка зависимостей
```bash
npm install
```

### Режим разработки (веб)
```bash
npm run dev
```

### Сборка веб-версии
```bash
npm run build
```

### Запуск в режиме Tauri (десктоп-приложение)
```bash
npm run tauri:dev
```

### Сборка десктопного установщика
```bash
npm run tauri:build
```
