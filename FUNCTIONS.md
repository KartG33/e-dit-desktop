# Документация функций форматирования E-dit

В данном документе подробно описаны все встроенные функции обработки текста, алгоритмы их работы, горячие клавиши и дополнительные аналитические инструменты.

---

## 1. Базовые команды форматирования (Basic)

Команды вкладки «Команды» направлены на очистку разметки, исправление опечаток пробелов и базовую типографику.

| Название функции | Описание | Алгоритм (Регулярное выражение / Код) |
| :--- | :--- | :--- |
| **Лишние пробелы** | Заменяет множественные пробелы и знаки табуляции на один пробел. | `.replace(/[ \t]{2,}/g, ' ')` |
| **NBSP** | Заменяет неразрывные пробелы (`\u00A0`) на стандартные пробелы. | `.replace(/\u00A0/g, ' ')` |
| **Обрезать края** | Удаляет все пробельные символы в начале и в конце каждой строки. | `.split('\n').map(line => line.trim()).join('\n')` |
| **ВЕРХНИЙ** | Переводит весь текст в верхний регистр. | `.toUpperCase()` |
| **нижний** | Переводит весь текст в нижний регистр. | `.toLowerCase()` |
| **Каждое Слово** | Переводит первую букву каждого слова в верхний регистр. | `.replace(/\b\w/g, char => char.toUpperCase())` |
| **Предложения** | Автоматически делает заглавной первую букву в начале каждого предложения. | Поддерживает Unicode через флаг `/gu` |
| **К знакам** | Прижимает знаки препинания `.,;:!?` к предшествующему слову (удаляет пробелы перед ними). | `.replace(/\s+([.,;:!?])/g, '$1')` |
| **От знаков** | Добавляет пробел после знаков препинания `.,;:!?`, если за ними следует непробельный символ. | `.replace(/([.,;:!?])(\S)/g, '$1 $2')` |
| **Без пунктуации** *(Опасная)* | Удаляет все знаки пунктуации из текста (точки, запятые, тире, скобки, кавычки и др.). | `.replace(/[.,;:!?'"«»()[\]{}\-—–]/g, '')` |
| **Без скобок** *(Опасная)* | Удаляет круглые `()` и квадратные `[]` скобки вместе со всем содержимым внутри них. | `.replace(/\([^)]*\)\|\[[^\]]*\]/g, '')` |
| **Без пустых строк** | Схлопывает тройные и более переносы строк в стандартный двойной абзац. | `.replace(/\n\s*\n\s*\n/g, '\n\n')` |
| **В одну строку** | Заменяет все переносы строк на пробелы, преобразуя текст в одну длинную строку. | `.replace(/\n+/g, ' ')` |
| **По предложениям** | Разбивает текст, перенося каждое новое предложение на новую строку. | `.replace(/([.!?])\s+/g, '$1\n')` |
| **Без нумерации** | Удаляет цифровую нумерацию списков в начале строк (например, `1.`, `2)`). | `.replace(/^\s*\d+[.)]\s*/gm, '')` |

## 2. Команды обработки текстов песен (Suno)

Специализированные команды для обработки структуры песен, генерации текстов для ИИ (например, Suno) и очистки от лишней разметки.

*   **Очистить теги:** Очищает теги настроения или стилей, оставляя только тип блока. Пример: `[Verse 1 | upbeat]` преобразуется в `[Verse 1]`.
*   **Отбивка тегов:** Находит все теги вида `[...]` и вставляет пустые строки до и после них, красиво структурируя текст на куплеты и припевы.
*   **Заглавные:** Автоматически переводит первую букву каждой строки в верхний регистр (при этом пропускает строки, которые являются тегами структуры `[...]`).
*   **Только текст:** Полностью удаляет все мета-теги `[...]` и лишние пустые строки, оставляя чистый текст песни.
*   **Структура:** Удаляет весь текст песни, оставляя только цепочку тегов `[...]` для анализа структуры композиции.
*   **Умные пробелы:** Сжимает множественные пустые строки внутри одного текстового блока, сохраняя аккуратные границы между куплетами.
*   **CAPS теги:** Переводит содержимое всех тегов `[...]` в верхний регистр для улучшения читаемости (например, `[verse 1]` -> `[VERSE 1]`).

## 3. Цепочки команд (Пресеты)

Пресеты позволяют объединять любые базовые, Suno или символьные команды в последовательные цепочки обработки. При активации пресета текст последовательно проходит через функции из цепочки.

*   **Создание пресета:** Через форму во вкладке «Пресеты» задается имя и выбираются команды.
*   **Удаление пресета:** Осуществляется через контекстное меню (клик правой кнопкой мыши по кнопке пресета) с последующим подтверждением.
*   **Хранение:** Все пресеты сохраняются в IndexedDB и доступны между сессиями.

---

## 4. Панель анализа и быстрого удаления символов

Панель **Символы** автоматически сканирует текст на наличие специальных символов (знаков препинания, скобок, разметки Markdown и др.):
1.  Символы сортируются по частоте их использования в тексте.
2.  Клик на кнопку любого символа в панели мгновенно удаляет все его вхождения во всём тексте.

---

## 5. Горячие клавиши (Shortcuts)

Для ускорения работы в редакторе поддерживаются стандартные клавиатурные комбинации:

*   `Ctrl + Z` — Отмена последнего действия (Undo)
*   `Ctrl + Shift + Z` / `Ctrl + Y` — Повтор отмененного действия (Redo)
*   `Ctrl + S` — Сохранить активный текст в файл (работает через системный диалог Tauri)
*   `Ctrl + O` — Открыть текстовый файл (TXT, MD, LYRICS) напрямую в активную панель
