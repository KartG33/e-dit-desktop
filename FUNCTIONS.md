# Документация функций форматирования E-dit

В данном документе подробно описаны все встроенные функции обработки текста, алгоритмы их работы, горячие клавиши, а также архитектурные особенности аналитических инструментов.

---

## 1. Базовые команды форматирования (Basic)

Команды вкладки **«Команды»** направлены на автоматическую очистку разметки, исправление пробелов вокруг знаков препинания и базовые регистровые преобразования.

| Название функции | Назначение и Описание | Алгоритм (Регулярное выражение / Код) |
| :--- | :--- | :--- |
| **Лишние пробелы** | Заменяет множественные пробелы и знаки табуляции на один пробел. | `.replace(/[ \t]{2,}/g, ' ')` |
| **NBSP** | Заменяет неразрывные пробелы (`\u00A0`) на обычные. | `.replace(/\u00A0/g, ' ')` |
| **Обрезать края** | Удаляет все пробельные символы в начале и конце каждой строки. | `.split('\n').map(line => line.trim()).join('\n')` |
| **ВЕРХНИЙ** | Переводит весь текст в верхний регистр. | `.toUpperCase()` |
| **нижний** | Переводит весь текст в нижний регистр. | `.toLowerCase()` |
| **Каждое Слово** | Делает первую букву каждого слова заглавной. | `.replace(/\b\w/g, char => char.toUpperCase())` |
| **Предложения** | Автоматически делает заглавной первую букву в начале каждого предложения. | `.replace(/(^\s*\p{L}\|[.!?]\s+\p{L})/gu, m => m.toUpperCase())` |
| **К знакам** | Прижимает знаки препинания `.,;:!?` к слову (удаляет пробелы перед ними). | `.replace(/\s+([.,;:!?])/g, '$1')` |
| **От знаков** | Добавляет ровно один пробел после знаков препинания `.,;:!?`, если далее идет текст. | `.replace(/([.,;:!?])(\S)/g, '$1 $2')` |
| **Без пунктуации** | Полностью вырезает из текста все знаки препинания, кавычки и тире. | `.replace(/[.,;:!?'"«»()[\]{}\-—–]/g, '')` |
| **Без скобок** | Удаляет круглые `()` и квадратные `[]` скобки вместе с их содержимым. | `.replace(/\([^)]*\(|\[[^\]]*\]/g, '')` |
| **Без пустых строк** | Схлопывает три или более идущих подряд переноса строки в два (стандартный абзац). | `.replace(/\n\s*\n\s*\n/g, '\n\n')` |
| **В одну строку** | Заменяет все переносы строк на обычные пробелы, вытягивая текст в линию. | `.replace(/\n+/g, ' ')` |
| **По предложениям** | Переносит каждое следующее предложение на новую строку. | `.replace(/([.!?])\s+/g, '$1\n')` |
| **Без нумерации** | Вырезает нумерованные списки в начале строк (например, `1.`, `4)`). | `.replace(/^\s*\d+[.)]\s*/gm, '')` |

> [!WARNING]
> Функции **«Без пунктуации»** и **«Без скобок»** помечены как опасные (`dangerous`), так как они безвозвратно удаляют части текста. Пользуйтесь ими с осторожностью или используйте функцию отмены (`Ctrl+Z`).

## 2. Команды обработки текстов песен (Suno)

Специализированные инструменты во вкладке **«Suno»** служат для структурирования метаданных и разметки (куплеты, припевы) в текстах песен, адаптируя их для генерации в нейросетях.

| Название функции | Описание действия | Пример преобразования |
| :--- | :--- | :--- |
| **Очистить теги** | Удаляет стили и описание настроения внутри тегов, сохраняя только имя блока. | `[Verse 1 \| upbeat]` $\rightarrow$ `[Verse 1]` |
| **Отбивка тегов** | Добавляет пустые строки до и после каждого блочного тега `[...]` для визуального разделения. | `[Chorus]` $\rightarrow$ `\n[Chorus]\n` |
| **Заглавные** | Делает первую букву каждой текстовой строки заглавной (игнорирует строки-теги `[...]`). | `hello world` $\rightarrow$ `Hello world` |
| **Только текст** | Вырезает все блочные теги в скобках `[...]` и схлопывает лишние переносы строк. | Оставляет чистые стихи / лирику. |
| **Структура** | Удаляет весь полезный текст лирики, оставляя на выходе только цепочку тегов. | `[Verse 1]\n[Chorus]` |
| **Умные пробелы** | Очищает множественные пустые строки внутри блоков текста, сохраняя структуру куплетов. | Схлопывает лишние разрывы внутри куплета. |
| **CAPS теги** | Переводит содержимое всех тегов `[...]` в верхний регистр для читаемости. | `[intro]` $\rightarrow$ `[INTRO]` |

## 3. Цепочки команд (Пресеты)

Пресеты позволяют объединять любые базовые, Suno или символьные команды в последовательные цепочки обработки. При активации пресета текст последовательно проходит через функции из цепочки.

*   **Создание пресета:** Через форму во вкладке **«Пресеты»** задается имя и выбираются команды.
*   **Удаление пресета:** Осуществляется через контекстное меню (клик правой кнопкой мыши по кнопке пресета) с последующим подтверждением.
*   **Хранение:** Все пресеты сохраняются в IndexedDB и доступны между сессиями.

## 4. Анализатор символов (Symbol Analyzer)

Панель **«СИМВОЛЫ»** автоматически сканирует текст на наличие специальных знаков, пунктуации, элементов разметки Markdown и прочих небуквенных символов.

### Логика работы и особенности:
1. **Динамический анализ:** Приложение собирает статистику использования спецсимволов и сортирует их по убыванию частоты встречаемости.
2. **Мгновенное удаление:** Каждая кнопка на панели символов является интерактивной. Нажатие на кнопку символа мгновенно удаляет все его вхождения из текста активного редактора.
3. **Оптимизация производительности (Debounce):** Чтобы исключить зависание интерфейса при непрерывном вводе текста, сканирование и обновление символьной панели выполняется с задержкой **300 мс** после остановки печати.
4. **Информативные подсказки (Tooltips):** Сами кнопки отображают только символ для сохранения компактности интерфейса. При наведении курсора на кнопку всплывает подсказка с количеством найденных символов (например: `Удалить все символы "-" (встречается: 24)`).

## 5. Горячие клавиши (Shortcuts)

Для быстрого управления процессом редактирования и взаимодействия с операционной системой в приложении предусмотрены клавиатурные сочетания:

*   `Ctrl + Z` — Отменить последнее изменение в активном редакторе (Undo).
*   `Ctrl + Shift + Z` или `Ctrl + Y` — Повторить ранее отмененное действие (Redo).
*   `Ctrl + S` — Сохранить активный текст в локальный файл через нативное диалоговое окно Tauri.
*   `Ctrl + O` — Выбрать и загрузить текстовый файл (`.txt`, `.md`, `.lyrics`) в активное окно редактора.
